# 💪 FitPartner Almaty

Платформа для поиска фитнес-партнёров в Алматы.

## Быстрый старт
```bash
cd backend
npm install
npm run dev
```

Открыть: http://localhost:5000

## Требования
- Node.js 16+
- MongoDB (локальный или Atlas)

## Структура
```
fitpartner/
├── backend/     ← Express + MongoDB API
└── frontend/    ← HTML/CSS/JS сайт
```
